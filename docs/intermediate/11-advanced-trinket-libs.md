# Advanced Python Libraries

!!! note "Historical reference"
    This page lists the libraries that were pre-installed in the old **Trinket** sandbox
    (now [deprecated](../about.md#a-note-on-platform-history)). Some of them — such as
    `processing` and `pygal` — were specific to Trinket and are not part of a standard
    local Python installation. Use `pip install <library>` to add the ones you need.

```py
import sys
import math
import matplotlib
import numpy
import processing
import pygal
import random
import re
import string
import time
import turtle
import urllib
```
